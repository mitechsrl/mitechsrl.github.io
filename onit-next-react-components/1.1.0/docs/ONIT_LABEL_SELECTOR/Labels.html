    1. OnitLabelSelector

    OnitLabelSelector

    Label selector component. Display an area where the user can add labels and search them from a provided list.

    import {OnitLabelSelector} from  '@mitech/onit-next-react-components/dist/labels/OnitLabelSelector'
<OnitLabelSelector
    createLabelText = { string }
    emptyErrorText = {string}
    labelGet = {someFunction}
    onChange = {someFunction}
>
</OnitLabelSelector>

    Props

    createLabelText String, label to be displayed in the create area

    emptyErrorText String, label to be displayed in the error area

    labelGet Async function function that will resolve with the labels list. Must return a [{id:string, label:string},...,{id:string, label:string}] array or reject

    labelPost Async function that will be called when a new label needs to be created. Must return a {id:string, label:string} object or reject

    labels Array of label objects. Each object is defined as

    {id: string, label:string}

    onChange Function, called on label selection change

    onError Function, called on error

    enableCreation boolean, Enable or disable the label creation area
